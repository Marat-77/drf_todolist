Django REST framework.

### Установка библиотеки django-cors-headers
для настройки политики CORS
```commandline
pip install django-cors-headers
```

### Установка React.js:
```commandline
npx create-react-app frontend
```
### Установка axios:
```commandline
cd frontend/
npm install axios
```

### Запуск тестового сервера React
```commandline
npm start
```
